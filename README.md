# Jogo-Test-Python
Jogo feito para teste prático


EVIL ARENA


Para se mover - A W S D ou SETAS

Objetivos 
- Fugir das assombrações durante tempo de jogo.
- pegue a cruz para ganhar HP e matar todos os monstros
- jogo sem ataque direto para encurtar código


Guia para instalação:

- Baixe o repositório
- vá até a pasta raiz
- rode via terminal python game.py
